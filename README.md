# Hi-Chat
A simple flutter chat app
ChatApp is a simple, reliable messaging app designed for fast real-time conversations. Sign in quickly with OTP or Google, create a profile, and start one-on-one chats instantly. keeping your contacts organized on the Home and Profile screens.
Built on a cloud backend for near-instant message delivery and smooth sync across devices, ChatApp focuses on usability: a clean interface, quick onboarding, and robust authentication flows (OTP and Google Sign-In supported). Ideal for personal chats, small communities, and teams looking for a lightweight alternative to bloated messengers.
We prioritize practical privacy and secure authentication; the app uses established authentication providers and cloud services to handle user accounts and message persistence.
Key Features:

Quick Sign-In: OTP and Google sign-in for fast account access.
Real-Time Chat: Instant one-on-one messaging and group conversations.
User Profiles: Personalize your profile and view contacts.
Simple, Fast UI: Minimal, responsive interface for smooth use.
